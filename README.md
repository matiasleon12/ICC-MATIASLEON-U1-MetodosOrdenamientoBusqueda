## 🛠️ Descripción

Este proyecto implementa y compara diferentes algoritmos de ordenamiento y busqueda en Java, incluyendo:
- Método Borbuja avanzado
- Método de insercion
- Método de Seleccion
- Método de busqueda binaria

---

## 🚀 Ejecución

Para ejecutar el proyecto:

1. Compila el código:
    ```bash
    controllers.java
    ```
2. Ejecuta la aplicación:
    ```bash
    java App
    ```
